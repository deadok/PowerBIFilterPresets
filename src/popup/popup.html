Limit site access for better privacy

For better privacy, consider limiting this extension's site access in Chrome to only your Power BI domains instead of allowing all sites.

Open Chrome's Extensions page, click Details on this extension, or right-click the extension icon and select Manage extension. Then set Site access to your specific Power BI report domains.

How to use presets

Quick instructions for the main popup workflows.

Save current filters

Open a Power BI report, set the filters you want, then click Save current filters. Review the captured filters and save them as a preset.

Apply a preset

Select a saved preset and click Apply preset. The extension tries to set the report filters to the saved values and shows any missing filters or values in the log.

Copy JSON

Select a preset and click Copy preset JSON to copy a shareable preset document for backup or sharing.

Add a preset

Click New preset to create a preset manually. You can update the JSON draft before saving.

Paste JSON

In New preset, click Paste JSON to insert a copied preset document from the clipboard. The extension validates it before saving.
